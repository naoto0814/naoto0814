## 📱 制作物

### [GymTimer](https://github.com/naoto0814/GymTimer)
**React Native / TypeScript**

筋トレのセット間インターバルを管理するタイマーアプリ。

**開発のきっかけ：**
既存のアプリでは種目ごとにタイマーを設定し直す手間や通知のたびに音楽が止まるといった小さなストレスがあり、それを解消するために自作しました。身近な課題を技術で解決する楽しさを実感したプロジェクトです。

**工夫した点・実装内容：**
- **技術選定と移行：** Webアプリとして試作した後、バックグラウンド対応や通知機能の必要性を感じ、React Nativeへの移行を判断して実装しました。
- **ユーザー体験の追求：** 自分が一番のユーザーとなり、日々の筋トレで感じる小さなストレスをコードで解決する実践的な開発を経験しました。
- **モバイル特有の挙動への対応：** `AppState` APIを用いたバックグラウンド移行時の正確な時間計算や、`expo-notifications` による画面オフ時の終了通知など、OS仕様を考慮した実装を行いました。
- **データのローカル永続化：** `AsyncStorage` でトレーニングメニューを保存し、アプリ再起動後もユーザーの設定を保持する仕組みを構築しました。
- **Reactの状態管理への理解：** `useState` や `useRef` の適切な使い分け、タイマー処理におけるクロージャ問題への対処を通じて、Reactの再描画の仕組みを実践の中で習得しました。
- **`useEffect`による責務の分離：** 処理を発火タイミングごとに `useEffect` で分割し、コードの見通しと保守性を向上させました。
